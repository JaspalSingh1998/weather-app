# Features

1. Data Vizualization using Chart.js 📉
2. Weather Update for current day and upcoming 4 days. 🌡
3. Created using Vanilla JavaScript 👩‍💻
4. Search City 🔎
5. Geolocation API for User location (with privacy in mind). 🔏
